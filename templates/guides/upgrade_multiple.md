# Changelog


# Upgrade Multiple Devices

## Description

The example shows how to upgrade multiple devices on a single Terraform run.

Devices are selected by their:

- location (site),
- role,
- hostname,
- hardware platform,
- IP address, etc.

## Run

```sh
terraform init
terraform apply
```

## Cleanup

```sh
terraform destroy
```
